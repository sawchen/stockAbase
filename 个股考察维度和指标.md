# 个股考察维度和指标

A股上市公司数量高达5500个。从庞大的股票群中甄选几只个股，条件必须绝对苛刻。

| 序号 | 考察维度和指标 |
|------|----------------|
| 1 | 个股资讯页面各种信息寥寥无几或网站很少更新 |
| 2 | 董秘连续2个月不回答投资者提问或总是答非所问、敷衍了事 |
| 3 | 连续24个月无重大利好消息（如无研报） |
| 4 | 无实控人（或实控人持股极少）或董事长本人不持股（或持股极少） |
| 5 | 董事长年龄不超过60岁（院士除外） |
| 6 | 前十大流通股东中知名机构投资者少于3个 |
| 7 | 主营业务不清晰（所涉行业过多过杂） |
| 8 | 连续三年亏损或多个季度增收不增利（利润率偏低） |
| 9 | 资产负债率 > 60%（控股股东高比例质押） |
| 10 | 融资余额与流通市值之比 > 10% |
| 11 | 近半年控股股东和管理层有减持行为且当前减持未完毕 |
| 12 | 股东人数近1-3个月显著增加 |
| 13 | 被司法机关或证监会立案查处或公司官司不断 |
| 14 | 所在行业的市场行情依旧偏弱 |
| 15 | 个股业绩增长或股价上涨的支撑逻辑比较模糊 |
| 16 | 公司生产端存在负面新闻（原料大幅涨价或断供，停产、产品不合格） |
| 17 | 个股或板块龙头股主要均线向下发散或 MACD 红柱明显变短甚至消失 |
| 18 | 个股或板块内标杆股快速大涨近3个月或持续上涨近6个月 |
| 19 | 年报季报中新增负面信息或疑点较多 |
| 20 | 国内外投行对所选个股或板块内龙头股的评级变得谨慎甚至消极 |
| 21 | 主管部门或主流媒体发表防止股市或板块过热的言论 |
| 22 | 个股所在行业统计数据不及预计（甚至令人十分意外） |
| 23 | 中美大国关系恶化 |
| 24 | 美股主要股指频繁剧烈波动（华尔街悲观看空情绪蔓延） |

---

**说明**：拟选个股或已买个股存在以上情形之一，即应拒选、调仓换股或干脆清仓。
