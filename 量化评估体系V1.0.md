# A股基本面量化评估体系 V1.0

## 评分规则
- 每项满分5分，最低0分
- 得分越高表示风险越低/质量越好
- 总分120分（24项×5分）

---

## 一、信息披露质量（3项）

### 1. 资讯更新频率（满分5分）
| 得分 | 标准 |
|------|------|
| 5分 | 近1个月有3条以上公告/新闻 |
| 4分 | 近1个月有1-3条公告/新闻 |
| 3分 | 近3个月有公告更新 |
| 2分 | 近6个月有公告更新 |
| 1分 | 近12个月有公告更新 |
| 0分 | 超过12个月无更新或无资讯 |

**数据来源**：东方财富网、同花顺、巨潮资讯

**量化方式**：爬取个股资讯页面更新频率

---

### 2. 董秘互动质量（满分5分）
| 得分 | 标准 |
|------|------|
| 5分 | 近2个月董秘回复10次以上，且平均字数>50字 |
| 4分 | 近2个月董秘回复5-10次，平均字数>30字 |
| 3分 | 近2个月董秘回复1-5次 |
| 2分 | 近6个月董秘有回复 |
| 1分 | 董秘偶尔回复且字数<20字（敷衍） |
| 0分 | 连续超过2个月无任何回复 |

**数据来源**：东方财富董秘互动平台、交易所互动易

**量化方式**：
- 爬取"投资者关系"董秘问答记录
- 统计回复频率、回复字数、是否答非所问
- "答非所问"定义：回复内容与问题关键词重合<30%

---

### 3. 研报覆盖频率（满分5分）
| 得分 | 标准 |
|------|------|
| 5分 | 近24个月研报数量≥20篇 |
| 4分 | 近24个月研报数量10-19篇 |
| 3分 | 近24个月研报数量5-9篇 |
| 2分 | 近24个月研报数量1-4篇 |
| 1分 | 近24个月无研报，但有券商提及 |
| 0分 | 连续24个月无任何研报覆盖 |

**数据来源**：东方财富研报中心、同花顺、Wind

**量化方式**：统计近24个月机构研报数量

---

## 二、股权结构（2项）

### 4. 实控人持股比例（满分5分）
| 得分 | 标准 |
|------|------|
| 5分 | 实控人直接或间接持股≥30% |
| 4分 | 实控人持股15%-30% |
| 3分 | 实控人持股5%-15% |
| 2分 | 实控人持股1%-5% |
| 1分 | 实控人持股<1%但仍为实控人 |
| 0分 | 无实控人或持股极少（<0.5%） |

**数据来源**：年报、半年报、公司公告

**量化方式**：F10股权结构穿透计算

---

### 5. 董事长持股与年龄（满分5分）
| 得分 | 标准 |
|------|------|
| 5分 | 董事长持股≥1%且年龄35-60岁 |
| 4分 | 董事长持股0.1%-1%或年龄35-60岁 |
| 3分 | 董事长持股<0.1%但年龄适中 |
| 2分 | 董事长年龄>65岁或<30岁 |
| 1分 | 董事长年龄>70岁 |
| 0分 | 董事长年龄<25岁或>65岁（特殊情况需说明） |

**数据来源**：年报、苏宁董秘互动、同花顺F10

**说明**：院士除外可加分，博士学历可放宽5岁

---

### 6. 机构持仓（满分5分）
| 得分 | 标准 |
|------|------|
| 5分 | 前十大流通股东中公募/私募/外资≥5家 |
| 4分 | 前十大流通股东中知名机构3-4家 |
| 3分 | 前十大流通股东中知名机构1-2家 |
| 2分 | 前十大流通股东全部为自然人或一般法人 |
| 1分 | 前十大流通股东中自然人持股>80% |
| 0分 | 前十大流通股东信息不透明或全部关联人 |

**知名机构定义**：汇金、证金、社保、公募基金（规模>50亿）、知名私募（管理规模>100亿）、外资（QFII、北上资金持股>1%）

**数据来源**：季报、年报公布的十大股东数据

## 实施建议

### 数据获取优先级
1. **高优先级**：年报季报（结构化数据）、交易所公开数据
2. **中优先级**：研报、董秘互动、舆情
3. **低优先级**：新闻舆情、美股波动（辅助）

### 实施路径
1. 先用年报/季报数据完成可量化维度（7、8、9、11、12、14）
2. 再接入舆情和董秘数据（1、2、17）
3. 最后接入技术面和外部数据（19、20、23、24）

### 注意事项
- 部分维度（如第2条答非所问）难以完全量化，需人工复核
- 建议设置权重：财务质量（30%）、监管风险（20%）、技术面（15%）、外部环境（15%）、其他（20%）
- 每季度更新评分，年报季重点复核
