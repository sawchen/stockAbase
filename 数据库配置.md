## 表结构

### 1. stock_info（股票基本信息表）

| 字段 | 类型 | 说明 |
|------|------|------|
| symbol | String | 股票代码，格式如 `600000.SH`、`000001.SZ` |
| name | String | 股票名称 |
| exchange | String | 交易所（SH/SZ） |
| market | String | 市场 |
| list_date | Nullable(String) | 上市日期，格式 `YYYY-MM-DD` |
| industry | String | 所属行业 |
| market_cap | Decimal(20,2) | 市值（元） |
| status | UInt8 | 状态（1=正常） |
| created_at | DateTime | 记录创建时间 |

- **记录数**：8,627 条
- **用途**：查询股票基本信息、行业分类、市值等

---

### 2. stock_minute（分钟行情数据表）【主力表】

| 字段 | 类型 | 说明 |
|------|------|------|
| symbol | String | 股票代码，格式 `600000.SH` |
| trade_date | Date | 交易日期 |
| trade_time | DateTime | 交易时间（含时区 `+08:00`） |
| trade_time_cst | DateTime | 中国标准时间（冗余字段） |
| open | Decimal(10,2) | 开盘价（元） |
| high | Decimal(10,2) | 最高价（元） |
| low | Decimal(10,2) | 最低价（元） |
| close | Decimal(10,2) | 收盘价（元） |
| volume | UInt64 | 成交量（股） |
| amount | Decimal(20,2) | 成交额（元） |
| created_at | DateTime | 记录创建时间 |

- **记录数**：233,018,327 条（约2.33亿）
- **时间范围**：2022-01-04 至 2026-04-03
- **覆盖股票数**：5,314 只
- **日均数据量**：约 310,000-320,000 条记录，4,140-4,180 只股票
- **数据频率**：5分钟K线（每个交易日 09:35-15:30，每5分钟一条）
- **用途**：A股历史分钟级行情查询，是本项目的核心数据表

## 查询示例

```python
import clickhouse_connect

client = clickhouse_connect.get_client(
    host='192.168.31.219',
    port=8123,
    username='user1',
    password='ckadmin2026',
    database='stock_data'
)

# 示例1：查询指定股票某日分钟数据
result = client.query("""
    SELECT trade_time, open, high, low, close, volume
    FROM stock_data.stock_minute
    WHERE symbol = '600000.SH'
      AND trade_date = '2026-03-15'
    ORDER BY trade_time
""")

# 示例2：查询股票基本信息
result = client.query("""
    SELECT symbol, name, industry, market_cap
    FROM stock_data.stock_info
    WHERE industry = '银行'
    LIMIT 10
""")

# 示例3：按日期范围统计每日数据量
result = client.query("""
    SELECT trade_date, count() as records, count(DISTINCT symbol) as stocks
    FROM stock_data.stock_minute
    WHERE trade_date >= '2026-03-01' AND trade_date <= '2026-03-31'
    GROUP BY trade_date
    ORDER BY trade_date
""")
```

## 注意事项

1. **股票代码格式**：查询时使用 `600000.SH`、`000001.SZ` 格式（带交易所后缀）
2. **时间字段**：trade_time 包含时区信息 `+08:00`，时区敏感场景需注意
3. **复权方式**：Baostock 数据默认后复权（adjustflag='2'），CK 数据与之对齐
4. **数据更新**：每个交易日 16:00 左右完成当日数据入库
