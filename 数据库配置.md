## 表结构

### stock_info（股票基本信息）
| 字段 | 类型 | 说明 |
|------|------|------|
| symbol | String | 股票代码 |
| name | String | 股票名称 |
| exchange | String | 交易所 |
| market | String | 市场 |
| list_date | Nullable(String) | 上市日期 |
| industry | String | 所属行业 |
| market_cap | Decimal(20,2) | 市值 |
| status | UInt8 | 状态 |
| created_at | DateTime | 创建时间 |

### stock_minute（分钟行情数据）
| 字段 | 类型 | 说明 |
|------|------|------|
| symbol | String | 股票代码 |
| trade_date | Date | 交易日期 |
| trade_time | DateTime | 交易时间 |
| open | Decimal(10,2) | 开盘价 |
| high | Decimal(10,2) | 最高价 |
| low | Decimal(10,2) | 最低价 |
| close | Decimal(10,2) | 收盘价 |
| volume | UInt64 | 成交量 |
| amount | Decimal(20,2) | 成交额 |
| created_at | DateTime | 创建时间 |
