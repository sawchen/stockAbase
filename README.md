# A股基本面量化评估系统

## 概述
基于24维度量化评估体系，对A股上市公司进行基本面评估打分。

## 快速开始

```bash
# 安装依赖
pip install -r requirements.txt

# 初始化数据库
clickhouse-client -h 192.168.31.219 -u user1 --password ckadmin2026 < sql/init.sql

# 评估单只股票
python -m src.cli --symbol sh.600000

# 评估并指定股票名
python -m src.cli --symbol sh.600000 --name 浦发银行
```

## 评估维度

共24个维度，总分120分：

| 类别 | 维度数 | 说明 |
|------|--------|------|
| 信息披露 | 3 | 资讯更新、董秘互动、研报覆盖 |
| 股权结构 | 3 | 实控人持股、董事长、机构持仓 |
| 业务清晰度 | 1 | 主营占比 |
| 财务质量 | 3 | 盈利、负债率、融资比例 |
| 股东行为 | 2 | 减持、股东人数 |
| 监管法律 | 2 | 立案、诉讼 |
| 行业景气 | 2 | 行业趋势、业绩逻辑 |
| 生产经营 | 2 | 供应链、产品竞争力 |
| 技术面 | 2 | 均线、MACD |
| 外部环境 | 3 | 年报风险、投行评级、政策舆论 |

## 数据来源

- 东方财富网
- 同花顺
- 巨潮资讯
- 交易所互动易
- 证监会官网
- 天眼查/企查查
- 行情数据（Yahoo Finance等）

## 评分规则

- 每项满分5分，最低0分
- 得分越高表示风险越低/质量越好
- 每项评分附带详细说明和数据来源URL
